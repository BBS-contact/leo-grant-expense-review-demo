LEO Grant Expense Review Dashboard v1.0

Local human-review dashboard for grant expense evidence signals. This interface reads local JSON outputs, shows rule traceability, records reviewer actions in browser localStorage, and does not perform donor compliance, fraud, legal, payment, supplier, production, registry, signing, or external execution decisions.
